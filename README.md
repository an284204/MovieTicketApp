App phân quyền thành 4 loại người dùng: user, staff, admin và boss.
Tương ứng với mỗi loại người dùng sẽ có các chức năng cơ bản của nó như: đăng kí, đăng nhập  bằng email, sdt, tìm kiếm, lọc phim, đặt chọn phim, vé, đồ ăn.
Staff kiểm tra trạng thái vé, hỗ trợ khách hàng. Admin quản lí phim, suất chiếu, phòng chiếu. Boss quản lí toàn bộ tài khoản hệ thống và thống kê doanh thu
